# Prefix notation 

To run it in interactive mode, go into racket, run it and type (main '()) and click Enter. Then type + 2 3 for example and it should return 5.0. 

To run it in bash mode, go into bash terminal. Navigate to the the zip folder  location. Type "racket prefix_calc.rkt -b" or "racket prefix_calc.rkt -batch" to run the file in batch mode. The type in the input. You can also run "racket.exe prefix_calc.rkt -batch" if it doesn't work.

devlog.md - Is the dairy entries

prefix_calc.bak - Is something that is added when racket file is ran

prefix_calc.rkt - main file that holds the prefix calculator
